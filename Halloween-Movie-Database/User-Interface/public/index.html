HALLOWEEN MOVIE DATABASE

Welcome to the Halloween Movie Database - OSU CS 340 Project.
This page is not the main page, but is here to meet the requirements for Project Step 03.


The Navigation Bar at the top of each page will take the user to different pages with different functions.
Some pages will have "DUMMY" Buttons that will take the user to a new page that willbe used to edit/update content.
Since this site is not connected to the database and no JS has been coded, the dummy buttons are used to display the edit pages.

Below you will find descriptions of what each page's eventual functionality will be once the back-end is incorporated.


Website Page Descriptions

  • Developer-Movie: For the Developer/Admin to Make Changes (Add, Delete, Update) to the Movies in the Database
  • Developer-Users: For the Developer/Admin to be able to manage the Users for the Halloween Movie Database.
  • Movie Search: User Interface to Search the Database for certain Movies.
  • Login/Create Account: For Users to Login and/or Create an Account.
    • Users: Can edit/manage personal infor, Watchlist, and Favorites.
    • Create Account: Used to set up a new Account.
